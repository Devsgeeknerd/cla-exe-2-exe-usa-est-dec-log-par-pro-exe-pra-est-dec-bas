# Exercício 2

***Conteúdo da Aula:***

Crie um código que irá verificar se um número é par ou ímpar, se o número for par, imprima o próprio número, se for ímpar, deverá imprimir a mensagem “Número não é par”!

O número é atribuído à variável `verificarPar`.
